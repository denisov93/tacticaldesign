Bad user interface design

Air heater

What is wrong with this image?

An air heater function is to generate heat! And so, in the freezing winters, we all love to turn it on to heat our rooms! It is Portable, heats the room with great efficiency, and low energy consumption.

The problem here is that when it heats and you try to move it you will burn yourself because of those tiny handles.

A possible solution would be, deepen the gaps in handles to get a better grip. A tiny detail will make it a lot better.

Good user interface design

Block note

Block Note is Perfection

A Block Note is simple, useful, easy to carry around.
It is writing friendly for left and right-handed users.
It takes less space than a normal notebook.
Be for quick notes, or class appointments, it always gets the job done
