Assignment 1

Bad user interface design

The Case:
Door blocked by toilet

Sometimes we expect certain things and often planning fails for various reasons.
We can see a path we want to go along, but we can't always see the way in which we want to get there, and turning back might not be an option.
That takes us to this case. 

Image: Desperate measures

The door doesn't open or close properly

We would think that having access to WC through a door is something considered trivial on our daily lives. Well not in this case...

A door provides access and privacy, as you can see the door was cut to allow the normal open and closing to provide access to the WC.

Some of the reasons this situation might have happen:
Toilet was replaced and the shape now interferes with the door.
Or when the toilet was placed the door was never tested for opening and closing or who did such job ignored it.

The solution for this case might be in,
switching the toilet with the drawer, due to the slimmer aspect of this drawer it seems to be a better fit and a proper solution. Of course this might include some pipe changing for water pumping and draining.

(Fun fact: this is not an isolated case)

Aspects: The good and bad

  • No waste of money
    Cutting the door to allow its function may not be ideal, but it is cheep.
  • Loss of privacy
    The door now has an unpleasant hole in it when closed, you can peek in!
  • Loss of aesthetic
    Owners could be ashamed by letting visitors in the house.
  • Huge design flaw
    Clearly not planned correctly. After the job was done some details where left untested.
  • Door replacement
    A new door can be placed but the problem would still remain. The cutting solution was not a good option.

Good user interface design

Nvidia's
GauGAN Engine

An interface was made for Nvidia's GauGAN Engine, an AI engine that converts a simple draw into a realistic like image, mostly used by researchers, the page showcases the potential of the engine with possibilities to be a Futuristic Computer Image Generator.
Can be accessed on the following url: GauGAN.

Image: Nvidia's GauGAN Interface

Image: Input and Output

Simple Design

Even without fully understanding the concept, we can judge the simplified design and its functions.
We have a brush, a paint bucket, a color picker and an undo arrow, which are common elements in image editing tools, and on the left side we have a list of elements we can draw with, we choose types instead of colors.
The output image is based on an image preset or a user provided image. The design is simple, we can guess what it does, like the question mark cube, doesn't it feel like it will randomly select an output image?
Since this is primarily intended for researchers, the colored or 3D animated buttons are not really a problem. It takes on the task of testing functions and keeping them simple, with the emphasis on the end results.
On the GauGAN page you can find the credits of the icon creators.

Aspects: The good and bad

  • Simple Design
    No fancy buttons, and it is ok!
  • Easy to use
    Commum icons of tools from image editing programmes make it easier to understand the functions.
  • Download option
    You can download your draw or the AI processed image, it is awesome.
  • Occasional bugs
    Sometimes drawing can cause malfunctions, which can be frustrating.
  • Undo button
    Unfortunately, the undo button goes back one action at most.
