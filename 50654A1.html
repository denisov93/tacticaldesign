Assignment 1

Bad user interface design

Elevator button panel

A confusing elevator button panel

When entering an elevator, one should be able to easily and quickly select the desired floor. This is not the case with this particular elevator button panel.

In an unfortunate attempt to make the design as compact as possible, this interface was made incredibly unpleasant to read. It’s confusing, since there doesn’t seem do be a pattern. The first number of each row is not indicative of anything. For example, if you try looking for the number “16” you may have to search longer than intended.

To make matters worse, it’s missing the number “13”, making it so there is even less of an organized pattern.

A simple fix, could be organizing each row so that it starts in a multiple of 5. Finding a number would be a much simpler task. As for the missing number, it could be omitted with an unclickable button, so the pattern wouldn’t be broken.

Good user interface design

Spotify's interface

Spotify's pleasent interface

Spotify’s user interface for Android is a prime example of a clean, well-executed design.

As one of the most popular music streaming apps, it fulfills its obligation of allowing for an easy, intuitive navigation. Whether a user intends on accessing their playlists, or search for a specific song or genre, all options are clearly listed, for a pleasant user experience.
